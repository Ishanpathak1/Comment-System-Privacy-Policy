Privacy Policy

Last updated: 10/24/2024

This privacy policy explains how the Comment System extension ("we", "our", or "us") collects, uses, and protects your information when you use the extension.

1. Information We Collect

The Comment System extension does not collect any personally identifiable information (PII). The extension operates locally on your browser and interacts with third-party services (e.g., blockchain smart contracts) without sending personal data to any servers controlled by us.

2. Permissions Used

The following permissions are requested by the extension to function properly:

  • activeTab: Allows the extension to inject the comment system into the active tab so that you can interact with web pages.
  • scripting: Needed to modify web pages to inject the comment system.
  • storage: Used to save user settings such as visibility of comments or interaction preferences.
  • Host permissions: Allows the extension to interact with web pages you visit to enable commenting functionality.

3. How We Use the Information

The information collected by the extension is solely used to enhance your browsing experience by enabling a commenting system for interacting with blockchain-based comments. No personal information is collected, stored, or transmitted.

4. Data Sharing

We do not share any data with third-party companies or services. The extension interacts with public blockchain networks where data such as comments are stored but are not associated with any personal identifiers.

5. Data Security

The extension does not store any personal information or sensitive data. All interactions are performed client-side, ensuring that data remains secure.

6. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page, and you are advised to review this page periodically for any updates.
